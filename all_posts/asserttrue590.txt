developer dave shea has released paintbrushjs , a lightweight image processing library that can apply a variety of filters to images on a web page under the covers , paintbrushjs uses the html canvas tag to implement its effects , automatically inserting canvas tags based on class names you can choose effects and control their parameters by adding attributes to various tags for example img src jordan jpg width height class filter blur data pb blur amount paintbrushjs works in any modern browser which means ie and below won t see the effects for a full list of effects available , check out the documentation or head over to the demo page
